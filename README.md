## Arquivo principal para editar

Abra o arquivo `config.js`. Nele você pode trocar textos, Instagram, WhatsApp, e-mail, CREF, cidade e caminhos das imagens.

Exemplo:

```js
imagens: {
    logo: "assets/images/logo.png",
    hero: "assets/images/guilherme-principal.webp",
    sobre: "assets/images/guilherme-sobre.webp",
    videoCapa: "assets/images/capa-video.webp"
}
```

## Como visualizar no computador

Você pode abrir o `index.html` diretamente no navegador. Para uma visualização mais fiel, use a extensão **Live Server** no VS Code.

## Como enviar para o GitHub

1. Faça uma cópia de segurança do projeto atual.
2. Crie uma branch nova no GitHub Desktop.
3. Copie todo o conteúdo desta pasta para a pasta clonada do repositório.
4. Faça o commit.
5. Clique em **Push origin**.
